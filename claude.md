# Claude Instructions for Test Analysis

## Project Context
This repository is used to analyze and generate test scenarios for the Parabank Banking Web Application.

Parabank is a demo online banking system that allows users to:

- Register new accounts
- Login to the banking portal
- View account balances
- Transfer funds between accounts
- Pay bills
- View transaction history

## Application Type
Web-based banking application.

The application contains:

- UI workflows for banking operations
- Customer account management
- Financial transactions
- Transaction history tracking

## Testing Objective
Analyze the application workflows and generate functional test scenarios.

Focus on:

- Core user journeys
- Positive test cases
- Negative test cases
Boundary conditions

## Key Workflows to Test

1. User Registration
2. Login
3. Account Overview

4. Fund Transfer
5. Bill Payment
6. Transaction History
